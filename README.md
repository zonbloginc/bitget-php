# bitget-php-sdk-api

## 使用说明

PHP版本: 7.4以上,通过composer进行管理依赖包，因此需要本机安装composer

```phpregexp
# 更新依赖
composer update

# 运行合约API示例
php example_mix.php

# 运行币币API示例
php example_mix.php

# 运行websocket示例
php example_ws.php start
```



本SDK基于 [api文档](https://bitgetlimited.github.io/apidoc/zh/swap/#25e54147de) 实现
